# Требования к системе
- Установленный Docker;
- Установленный .NET Sdk 10.0;
- Наличие терминала (Terminal для MacOS/Linux; PowerShell или CMD для Windows).
---
# Требования к ОС
### Проект поддерживает операционные системы для ПК, такие как: Windows; Linux; MacOS.
---
# Инструкция по запуску проекта
### Развёртывание базы данных и ASP.NET WebApi приложения
Для первого запуска потребуется запустить docker-compose из дериктории проекта. 
- Команда для запуска: `docker-compose up`

__После выполнения команды в консоли должна отобразиться информация об успешном поднятии контейнеров.__

### Миграция базы данных
После поднятия docker-контейнеров требуется произвести миграцию базы данных.
- Команда запуска для MacOS/Linux: `DbMigrationService/bin/Debug/net10.0/DbMigrationService`
- Команда запуска для Windows: `DbMigrationService/bin/Debug/net10.0/DbMigrationService.exe`

__После выполнения команды в консоли должна отобразиться информация об успешной миграции.__
---
### В результате выполнения вышеописанных действий - проект будет развёрнут и готов к использованию.
